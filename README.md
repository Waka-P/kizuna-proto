# きずなプロト（kizuna-proto）

子ども食堂運営者と物資提供者（企業・農家・個人）のマッチング用プロトタイプです。
`index.html` をブラウザで開くだけで利用できます。

## 実装機能

- ログイン機能（暫定）
  - 表示名 + 区分選択で簡易ログイン
- モード切替（子ども食堂 / 提供者）
  - モーダルでロール選択し切替（Period._dev のロール切替UIに寄せた実装）
- 子ども食堂モード
  - ニーズ投稿
  - 余剰物資一覧の閲覧
  - チャット（ファイル添付可）
- 提供者モード
  - 余剰物資投稿
  - 子ども食堂掲示板の閲覧
  - チャット（ファイル添付可）

## 使い方

### HTMLを開くだけで使う（サーバー不要）

`index.html` をブラウザで直接開いてください。

- 例: エクスプローラーで `index.html` をダブルクリック
- データはブラウザの `localStorage` に保存
- 同一ブラウザ内（同PCで複数タブ）でデータが同期されます

## 技術構成

- フロントエンド: HTML + CSS + JavaScript（画面ごとに分割）
- データ保存: `localStorage`（ブラウザ内）
- 外部DB: なし（JSON入出力で擬似DB運用）

### ファイル構成（主要）

- `index.html`: ログインページ
- `board.html`: 一覧ページ
- `post-detail.html`: 投稿詳細ページ（チャット導線あり）
- `post.html`: 投稿ページ
- `chat.html`: チャット一覧ページ
- `chat-room.html`: 1対1チャット詳細ページ
- `settings.html`: 設定ページ
- `shared.js`: 共通ロジック（状態管理・ユーティリティ）
- `styles.css`: スタイル読み込みハブ（`styles/` 以下を `@import`）
- `styles/`: CSS分割ファイル（`foundation.css` / `components.css` / `navigation.css` / `chat.css` / `chat-room.css`）
- `scripts/`: 各ページ専用スクリプト

## 備考

- ファイル添付は Base64 で送受信しており、プロトタイプ向けです。
- 本番運用時は認証強化、永続DB、オブジェクトストレージ（S3等）への移行が必要です。
